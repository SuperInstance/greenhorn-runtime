# The Dojo + Runtime Connection

The dojo (greenhorn) is where agents **learn**. The runtime (greenhorn-runtime) is where agents **execute**.

A vessel is a repo with structure. The runtime reads the vessel, executes tasks, pushes results. The vessel persists context between executions.

The dojo model: agents produce value while learning. The runtime: agents execute reliably across hardware. Together: agents that grow AND execute anywhere.

See: github.com/SuperInstance/greenhorn (dojo model)
