# greenhorn-runtime

## What It Is
Portable agent runtime in Go, C, C++, CUDA, Rust, Zig, Java, JS, Python.
Plants agents anywhere within hardware/API limits.

## The Problem It Solves
You have an agent. You need it to run on a VPS, a Jetson, a Raspberry Pi, a laptop. The greenhorn-runtime is the portable deployment target — clone the vessel repo, run the agent, push results.
